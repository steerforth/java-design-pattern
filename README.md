java-design-pattern
===================
面向对象设计原则:
--------------
*单一职责原则  
*开闭原则  
*里氏替换原则  
*依赖倒置原则  
*接口分离原则  
*米特尔原则

面向对象设计模式中:类与类之间总共有6种关系
---------------------
依赖、关联、聚合（整体与部分可以分开）、组合（部分对象与整体对象之间具有同生共死的关系）、继承、实现
